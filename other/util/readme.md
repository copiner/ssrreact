转换为十进制
```
1234[10进制]
1000+200+30+4=1*10^3+2*10^2+3*10^1+4*10^0=1234

1011[2进制]
1*2^3+0*2^2+1*2^1+1*2^0=8+0+2+1=11

1011[8进制]
1*8^3+1*8^1+1*8^0=512+8+1=521

1011[16进制]
1*16^3+1*16^1+1*16^0=4096+16+1=4113
```

十进制到十六进制
```
1610[10进制]

1610/16=100……10(A)；
100 /16= 6……4；
6 /16= 0……6；

故：1610(10)=64A(16).
```
十进制到八进制
```
2456 转化成八进制数字：4630

2456/8=307，余0；
307/8=38，余3；
38/8=4，余6；
4/8=0，余4。

将所有余数倒序相连，得到结果：4630
因此十进制的2456转换为八进制结果为4630
```
